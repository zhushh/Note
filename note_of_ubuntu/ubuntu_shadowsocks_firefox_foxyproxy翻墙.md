Shadowsock安装
----

Ubuntu安装shadowsocks
```
$ sudo apt-get install python-pip
$ sudo pip install shadowsocks
```
CentOS安装shadowsocks
```
$ sudo yum install python-setuptools && easy_install pip
$ sudo pip install shadowsocks
```
